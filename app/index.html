Create an Agreement in 3 Steps

Step 1: Upload a document

Simply upload the document or contract that you wish to sign. It will be stored on IPFS, a public decentralized filesystem

Step 2: Sign the document

You will need MetaMask or a similar ethereum enabled browser in order to complete this step. Clicking "I agree" should open MetaMask and request you to submit a transaction for zero ether. There will be a gas cost for this transaction, likely between $0.3 and $0.30.

Step 3: Invite others to sign the document

You will be given a URL that you can share with other people. When they use this URL they will be able to add their signature to the document in the same way. You can return to this URL yourself to see which ethereum addresses have added their signature.
